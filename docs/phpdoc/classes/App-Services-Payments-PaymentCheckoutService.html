\App\Services\Payments PaymentCheckoutService

Orchestrates starting a checkout session for a {@see Payment} using the configured gateway (Stripe or PayPal).

Also exposes helpers for storefront configuration: which methods are available, simulated checkout in debug, and credential hints for the UI.
